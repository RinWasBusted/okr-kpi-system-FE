# OKR/KPI System - Frontend

A web application for managing Objectives and Key Results (OKR) and Key Performance Indicators (KPI), built with React and Vite.

## Overview

The OKR/KPI system helps organizations manage strategic goals, track progress, and evaluate performance efficiently.

## Tech Stack

- **React** - JavaScript UI library
- **Vite** - Modern build tool
- **ESLint** - Code quality tool

## Requirements

- Node.js >= 14.0
- npm or yarn

## Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/     # Reusable components
├── pages/          # Main pages
├── routers/        # Routing configuration
├── services/       # API services
├── hooks/          # Custom hooks
├── utils/          # Utility functions
├── assets/         # Images, fonts, ...
├── App.jsx         # Root component
└── main.jsx        # Entry point
```

## Features

- OKR management (Objectives and Key Results)
- KPI tracking (Key Performance Indicators)
- Analytics dashboard
- Progress reports
